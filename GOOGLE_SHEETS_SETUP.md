# 📊 Настройка Google Sheets интеграции

Бот может синхронизировать твои покупки с Google Sheets и Excel. Вот пошаговая инструкция.

---

## 🔑 Шаг 1: Создание Google Cloud проекта

1. Перейди на [Google Cloud Console](https://console.cloud.google.com)
2. Нажми на логотип → "Выбрать проект" → "Создать проект"
3. Введи имя: `drink-tracker-bot`
4. Нажми "Создать"
5. Дождись загрузки проекта

---

## 📋 Шаг 2: Включение Sheets API

1. В левом меню перейди в "API и сервисы" → "Библиотека"
2. Найди "Google Sheets API"
3. Нажми "Включить"
4. Дождись загрузки

---

## 🔐 Шаг 3: Создание Service Account

1. Перейди в "API и сервисы" → "Учётные данные"
2. Нажми "Создать учётные данные" → "Service Account"
3. Введи:
   - **Имя сервисного аккаунта:** `drink-tracker-bot`
   - **ID:** заполнится автоматически
   - **Описание:** `Bot for tracking drink purchases`
4. Нажми "Создать и продолжить"
5. На следующем экране пропусти (нажми "Продолжить")
6. Ещё раз пропусти ("Готово")

---

## 🔑 Шаг 4: Создание и скачивание ключа

1. Перейди в "API и сервисы" → "Учётные данные"
2. Найди Service Account, который ты только что создал
3. Нажми на его почту
4. Перейди на вкладку "Ключи"
5. Нажми "Добавить ключ" → "Создать новый ключ"
6. Выбери "JSON"
7. Нажми "Создать"
8. Файл `credentials.json` автоматически скачается

---

## 📊 Шаг 5: Создание Google Sheet

1. Перейди на [Google Sheets](https://sheets.google.com)
2. Нажми "Создать" → "Таблица"
3. Введи название: `Cola_Tracker_Sync`
4. Скопируй ID из URL:
   ```
   https://docs.google.com/spreadsheets/d/PASTE_THIS_ID/edit#gid=0
                                           ^^^^^^^^^^^^^^^^^
   ```

---

## 🔗 Шаг 6: Предоставление доступа

1. Открой `credentials.json` (текстовый редактор)
2. Найди поле `client_email` (выглядит как почта)
3. Скопируй эту почту
4. Откройте Google Sheet
5. Нажми кнопку "Поделиться" (правый верхний угол)
6. Введи скопированную почту
7. Дай права "Редактор"
8. Отправь приглашение

---

## 📁 Шаг 7: Подготовка файлов

1. Скопируй скачанный `credentials.json` в папку `drink_tracker_bot/`
2. Отредактируй `.env` файл:

```env
GOOGLE_SHEETS_CREDENTIALS_JSON=./credentials.json
GOOGLE_SHEETS_ID=1BxiMVs0XRA5nFMKUVTAEZtOWaOMSEP41-9mArKpqCk4
```

Замени `GOOGLE_SHEETS_ID` на ID твоей таблицы из Шага 5.

---

## ✅ Проверка

Теперь бот может синхронизировать покупки! 

Когда ты нажмёшь "Синхронизировать" в меню администратора:
1. Бот получит все твои покупки из Supabase
2. Обновит Google Sheet
3. Файл появится в твоём Google Drive

---

## 🐛 Если не работает

### ❌ "Ошибка аутентификации"
- Проверь что `credentials.json` в правильной папке
- Убедись что Service Account имеет доступ к Sheet

### ❌ "Таблица не обновляется"
- Проверь что ID таблицы правильный в `.env`
- Убедись что Service Account имеет права "Редактор"
- Посмотри логи: `tail -f logs/bot.log`

### ❌ "Файл credentials.json не найден"
```bash
# Убедись что он в папке drink_tracker_bot
ls -la credentials.json
```
